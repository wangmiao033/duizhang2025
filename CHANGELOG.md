# 更新日志

## [最新版本] - 2026-01-26

### 🎉 新增功能

#### 📄 PDF发票导入功能
- **智能文件名解析**：支持从PDF文件名自动提取发票信息
  - 支持格式：`销售方+购买方+金额+日期.pdf`
  - 示例：`深圳龙魂+广州熊动22557.99+20260126.pdf`
  - 自动解析：发票抬头、金额、开票日期、状态等信息
- **快速录入**：新增"快速录入"按钮，支持手动输入文件名进行解析
- **PDF文件支持**：导入功能现在支持 `.json` 和 `.pdf` 文件格式

#### ✅ 发票核销功能
- **核销管理**：支持将发票与对账记录进行关联核销
- **核销状态显示**：发票列表中显示核销状态（已核销/未核销）
- **批量核销**：支持一条发票关联多条对账记录
- **核销统计**：显示已核销的记录数量和核销总金额
- **核销对话框**：友好的核销界面，支持多选对账记录

### 🎨 界面优化

- **发票列表增强**：新增"核销状态"列，清晰显示每条发票的核销情况
- **核销按钮**：每条发票记录新增"核销"按钮，操作更便捷
- **状态标签**：优化核销状态标签样式，视觉更清晰

### 📝 使用说明

#### PDF发票导入步骤：
1. 点击"导入 (JSON/PDF)"按钮
2. 选择PDF文件（文件名需包含：销售方+购买方+金额+日期）
3. 系统自动解析并填充表单
4. 补充税号后点击"保存发票记录"

#### 发票核销步骤：
1. 在发票列表中找到需要核销的发票
2. 点击"核销"按钮
3. 在弹出对话框中选择要关联的对账记录（可多选）
4. 确认核销，系统自动关联发票和对账记录
5. 发票列表会显示核销状态和核销记录数量

### 🔧 技术改进

- 优化ConfirmDialog组件，支持React元素作为消息内容
- 改进文件名解析算法，提高解析准确性
- 增强表单验证和错误提示
