# 更新说明

## 📢 最新更新 - 2026年1月26日

### 🎯 本次更新亮点

#### 1. PDF发票智能导入 📄
现在可以直接上传PDF发票文件，系统会智能解析文件名并自动填充发票信息！

**使用方法：**
- 点击"导入 (JSON/PDF)"按钮
- 选择PDF文件（文件名格式：`销售方+购买方+金额+日期.pdf`）
- 系统自动解析并填充表单
- 补充税号后保存即可

**示例文件名：**
```
深圳龙魂+广州熊动22557.99+20260126.pdf
```

**解析结果：**
- ✅ 发票抬头：广州熊动
- ✅ 金额：22557.99
- ✅ 开票日期：2026-01-26
- ✅ 状态：已开
- ✅ 备注：销售方：深圳龙魂

#### 2. 发票核销功能 ✅
新增发票核销功能，可以将发票与对账记录进行关联，方便追踪和管理！

**核心功能：**
- 🔗 **关联核销**：一条发票可以关联多条对账记录
- 📊 **状态显示**：清晰显示每条发票的核销状态
- 💰 **金额统计**：自动计算核销总金额
- 📋 **记录追踪**：查看每条发票关联了哪些对账记录

**操作步骤：**
1. 在发票列表中找到需要核销的发票
2. 点击"核销"按钮
3. 在弹出的对话框中选择要关联的对账记录（可多选）
4. 确认后完成核销

**核销状态说明：**
- 🟡 **未核销**：黄色标签，表示尚未关联对账记录
- 🔵 **已核销**：蓝色标签，显示已关联的记录数量

## 💡 使用技巧

### 快速录入发票
如果PDF文件名格式正确，系统会自动解析。如果文件名格式不标准，可以使用"快速录入"功能：
1. 点击"快速录入"按钮
2. 输入或粘贴文件名
3. 系统解析后自动填充表单

### 批量核销
一条发票可以关联多条对账记录，适合以下场景：
- 一张发票对应多个月份的对账
- 一张发票对应多个游戏的对账
- 一张发票对应多个合作方的对账
