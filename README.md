# 对账管理系统

一个现代化的对账管理网站，支持研发对账、渠道对账、发票管理、客户管理等全流程业务管理，并提供丰富的数据分析和导出功能。

## ✨ 核心功能特性

### 📋 对账管理
- **研发对账**：完整的研发对账记录管理，支持结算金额自动计算
- **渠道对账**：渠道对账记录管理，支持按渠道和游戏分组查看
- **批量操作**：支持批量编辑、批量删除、批量导出
- **数据验证**：自动检测数据完整性和准确性

### 📄 发票管理
- **发票录入**：支持手动录入和PDF文件导入
- **智能解析**：从PDF文件名自动提取发票信息（销售方+购买方+金额+日期）
- **发票核销**：将发票与对账记录关联，支持一条发票关联多条记录
- **核销状态**：清晰显示每条发票的核销状态和核销金额
- **快速录入**：支持从文件名快速解析发票信息

### 👥 客户与合作伙伴管理
- **客户管理**：完整的客户信息管理，包括公司信息、联系方式、发票信息
- **快递中心**：快递记录管理，关联客户信息
- **标签管理**：为记录添加标签，方便分类和筛选

### 📊 数据分析与报表
- **数据统计**：实时显示记录总数、结算总额、游戏流水等关键指标
- **统计图表**：多种图表展示数据趋势和分布
- **高级图表**：更深入的数据可视化分析
- **项目利润分析**：项目级别的利润分析报表
- **数据对比**：多维度数据对比分析
- **统计报告**：生成详细的统计报告

### 📥 导入导出功能
- **Excel导入**：支持从Excel文件批量导入对账记录
- **Excel导出**：导出标准格式的对账单（Excel格式）
- **CSV导出**：导出CSV格式数据，方便其他系统使用
- **PDF导出**：导出PDF格式对账单
- **JSON导入导出**：支持数据备份和恢复
- **打印功能**：一键打印对账单

## 快速开始

### 基本工作流程

1. **添加对账记录**：
   - 进入"研发对账"标签页
   - 填写对账信息（结算月份、合作方、游戏、流水等）
   - 系统自动计算结算金额
   - 点击"添加记录"保存

2. **录入发票**：
   - 进入"发票"标签页
   - 方式一：手动填写发票信息
   - 方式二：上传PDF文件（文件名格式：`销售方+购买方+金额+日期.pdf`）
   - 系统自动解析并填充表单
   - 补充税号后保存

3. **核销发票**：
   - 在发票列表中找到需要核销的发票
   - 点击"核销"按钮
   - 选择要关联的对账记录（可多选）
   - 确认核销

4. **导出对账单**：
   - 在页面底部选择导出格式
   - Excel导出：标准格式的对账单
   - CSV导出：CSV格式数据
   - PDF导出：PDF格式对账单
   - 打印：直接打印

### 快捷键

- `Ctrl + F`：聚焦搜索框
- `Ctrl + P`：打印对账单
- `Ctrl + Enter`：保存编辑（在编辑模式下）
- `Esc`：关闭对话框/取消编辑

### 数据字段说明

#### 研发对账记录
- **结算月份**：对账所属月份（如：2025年9月）
- **合作方**：合作方名称（必填）
- **游戏**：游戏名称（必填）
- **游戏流水**：游戏流水金额（必填）
- **测试费**：测试费用
- **代金券**：代金券金额
- **通道费率**：通道费率百分比
- **税点**：税点百分比
- **分成比例**：分成比例百分比
- **折扣**：折扣系数
- **退款**：退款金额
- **结算金额**：自动计算（根据公式）

#### 发票信息
- **发票抬头**：发票抬头（必填）
- **税号**：纳税人识别号（必填）
- **开票金额**：发票金额
- **开票日期**：发票开具日期
- **状态**：未开/已开/作废
- **备注**：其他备注信息

#### 客户信息
- **公司名称**：客户公司名称
- **联系方式**：联系电话、邮箱等
- **地址信息**：公司地址
- **发票信息**：发票抬头、税号等
- **银行信息**：银行名称、账号等
