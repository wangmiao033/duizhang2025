# 对账管理系统

一个现代化的对账管理网站，支持游戏、游戏流水和代金券的数据管理，并可以导出Excel格式的对账单。

## 功能特性

- 🎮 **游戏管理**：添加和管理游戏信息（名称、平台、类型）
- 💰 **流水管理**：记录游戏流水（收入/支出）
- 🎫 **代金券管理**：管理代金券信息和使用状态
- 📊 **数据统计**：实时显示游戏总数、流水总额、代金券总额
- 📥 **导出功能**：一键导出Excel格式的对账单，包含所有数据表和汇总信息
- 💾 **数据持久化**：数据自动保存到浏览器本地存储

## 技术栈

- React 18
- Vite
- XLSX (Excel导出)
- Day.js (日期处理)

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用说明

1. **添加数据**：
   - 点击"添加游戏"标签页，填写游戏信息
   - 点击"添加流水"标签页，记录游戏流水
   - 点击"添加代金券"标签页，添加代金券信息

2. **查看数据**：
   - 在右侧表格区域切换不同标签页查看各类数据
   - 可以删除不需要的记录

3. **导出对账单**：
   - 点击底部的"导出对账单"按钮
   - 系统会自动生成包含所有数据的Excel文件
   - 文件包含4个工作表：游戏列表、游戏流水、代金券、汇总

## 数据字段说明

### 游戏
- 游戏名称（必填）
- 平台
- 游戏类型

### 游戏流水
- 游戏名称（必填）
- 日期
- 金额（必填）
- 类型（收入/支出）
- 备注

### 代金券
- 游戏名称（必填）
- 代金券代码
- 金额（必填）
- 日期
- 状态（未使用/已使用/已过期）
