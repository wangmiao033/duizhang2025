# 对账管理系统

一个现代化的对账管理网站，支持研发对账、渠道对账、发票管理、客户管理等全流程业务管理，并提供丰富的数据分析和导出功能。

## ✨ 核心功能特性

### 📋 对账管理
- **研发对账**：完整的研发对账记录管理，支持结算金额自动计算
- **渠道对账**：渠道对账记录管理，支持按渠道和游戏分组查看
- **批量操作**：支持批量编辑、批量删除、批量导出
- **数据验证**：自动检测数据完整性和准确性

### 📄 发票管理
- **发票录入**：支持手动录入和PDF文件导入
- **智能解析**：从PDF文件名自动提取发票信息（销售方+购买方+金额+日期）
- **发票核销**：将发票与对账记录关联，支持一条发票关联多条记录
- **核销状态**：清晰显示每条发票的核销状态和核销金额
- **快速录入**：支持从文件名快速解析发票信息

### 👥 客户与合作伙伴管理
- **客户管理**：完整的客户信息管理，包括公司信息、联系方式、发票信息
- **快递中心**：快递记录管理，关联客户信息
- **标签管理**：为记录添加标签，方便分类和筛选

### 📊 数据分析与报表
- **数据统计**：实时显示记录总数、结算总额、游戏流水等关键指标
- **统计图表**：多种图表展示数据趋势和分布
- **高级图表**：更深入的数据可视化分析
- **项目利润分析**：项目级别的利润分析报表
- **数据对比**：多维度数据对比分析
- **统计报告**：生成详细的统计报告

### 📥 导入导出功能
- **Excel导入**：支持从Excel文件批量导入对账记录
- **Excel导出**：导出标准格式的对账单（Excel格式）
- **CSV导出**：导出CSV格式数据，方便其他系统使用
- **PDF导出**：导出PDF格式对账单
- **JSON导入导出**：支持数据备份和恢复
- **打印功能**：一键打印对账单

### 🛠️ 实用工具
- **搜索筛选**：强大的搜索和筛选功能
- **排序功能**：支持多字段排序
- **历史记录**：操作历史记录，支持恢复
- **模板管理**：账单模板保存和加载
- **快速填充**：常用数据快速填充
- **数据备份**：完整的数据备份和恢复功能
- **提醒事项**：重要事项提醒管理
- **日历功能**：日期选择和查看

### 🎨 用户体验
- **响应式设计**：完美适配桌面和移动设备
- **暗色模式**：支持明暗主题切换
- **快捷键支持**：常用操作快捷键
- **拖拽排序**：支持拖拽调整记录顺序
- **复制记录**：快速复制已有记录
- **数据验证**：实时数据验证和错误提示
- **通知中心**：操作通知和消息提醒
- **用户指南**：内置使用帮助和指南

### 💾 数据管理
- **自动保存**：数据自动保存到浏览器本地存储
- **数据持久化**：所有数据本地持久化保存
- **数据安全**：数据备份和恢复机制

## 技术栈

- React 18
- Vite
- XLSX (Excel导出)
- Day.js (日期处理)

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用说明

1. **添加数据**：
   - 点击"添加游戏"标签页，填写游戏信息
   - 点击"添加流水"标签页，记录游戏流水
   - 点击"添加代金券"标签页，添加代金券信息

2. **查看数据**：
   - 在右侧表格区域切换不同标签页查看各类数据
   - 可以删除不需要的记录

3. **导出对账单**：
   - 点击底部的"导出对账单"按钮
   - 系统会自动生成包含所有数据的Excel文件
   - 文件包含4个工作表：游戏列表、游戏流水、代金券、汇总

## 数据字段说明

### 游戏
- 游戏名称（必填）
- 平台
- 游戏类型

### 游戏流水
- 游戏名称（必填）
- 日期
- 金额（必填）
- 类型（收入/支出）
- 备注

### 代金券
- 游戏名称（必填）
- 代金券代码
- 金额（必填）
- 日期
- 状态（未使用/已使用/已过期）
